01 chopratejas/headroom Python
⭐ 42.3k ▲ 14,982

LLM 上下文压缩层——在工具输出、日志、文件、RAG 分块进入大模型之前智能压缩，节省 60-95% Token，答案质量不变。

核心功能
  • 6 种压缩算法智能路由：JSON 专用 SmartCrusher、AST 感知代码压缩、基于 ModernBERT 的文本压缩等
  • CCR 可逆压缩：原始数据不丢失，通过 Compress-Cache-Retrieve 在需要时完整取回
  • 四种集成形态：Library / Proxy / Agent Wrap / MCP Server，零代码改动即可接入 Claude Code、Cursor 等主流工具
适用场景
AI 应用开发者 API 成本敏感用户 Agent 构建者 RAG 系统优化
02 Panniantong/Agent-Reach Python
⭐ 36.1k ▲ 8,483

赋予 AI Agent「全网视觉」——一行 CLI 即可让 Agent 读取并搜索 Twitter、Reddit、YouTube、GitHub、Bilibili、小红书，零 API 费用。

核心功能
  • 跨平台统一搜索接口：覆盖 6+ 主流社交与内容平台，无需申请各平台 API Key
  • CLI 原生集成：通过命令行直接与 AI 编程助手（Claude Code、Cursor 等）协同工作
  • 结构化输出：搜索结果以 Agent 友好格式返回，便于下游分析链路消费
适用场景
信息聚合 Agent 舆情监控 竞品调研 内容创作者
03 addyosmani/agent-skills JavaScript
⭐ 64.3k ▲ 6,332

生产级 AI 编码 Agent 工程技能集——由 Google 工程师 Addy Osmani 维护，覆盖代码审查、测试策略、安全审计、文档生成等标准化技能模板。

核心功能
  • 开箱即用的技能模板：代码审查规范、依赖更新流程、安全漏洞检查、自动文档生成等
  • 多平台兼容：Claude Code、Cursor、Copilot、Windsurf 等主流 AI 编程工具均可使用
  • Fork 即用：每个技能为标准化配置文件，按团队规范微调即可复用，降低从零定义规则的投入
适用场景
工程负责人 团队规范建设 AI 辅助开发 代码质量提升
04 Lum1104/Understand-Anything TypeScript
⭐ 64.5k ▲ 6,027

将任意代码库转化为可探索、可搜索、可问答的交互式知识图谱——让人类开发者和 AI Agent 都能快速理解代码结构。

核心功能
  • 代码库知识图谱化：自动扫描导入关系、模块依赖、配置关联，生成持久化图谱制品
  • 语义搜索与问答：在可视化图谱中直接搜索并提问，无需通读源文件

05 DeusData/codebase-memory-mcp C
⭐ 9.6k ▲ 5,419

高性能代码情报 MCP 服务器——毫秒级索引任意代码库为持久化知识图谱，支持 158 种语言，节省 99% Token 消耗，单二进制零依赖。

核心功能

  • 158 种语言支持：覆盖主流及小众编程语言的代码结构分析
